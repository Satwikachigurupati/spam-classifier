#  Spam Email Classifier
This is a simple Machine Learning project that detects whether a message is **Spam or Not Spam**.
##  What it does
- Takes SMS messages as input
- Converts text into numbers
- Trains a model to classify spam
- Predicts new messages
##  Model Used
- Naive Bayes / Logistic Regression / Linear SVM
##  Dataset
- SMS Spam dataset (loaded from GitHub link)
##  Tools Used
- Python
- Pandas
- Scikit-learn
- Google Colab
